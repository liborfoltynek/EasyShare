## Úplný popis (max 4000 znaků)
```
EasyShare je aplikace pro sdílení souborů přes váš vlastní webový server. Nahrajte jakýkoli soubor a okamžitě získáte krátký odkaz ke sdílení — bez cloudových služeb, bez omezení velikosti, s plnou kontrolou nad vašimi daty.

📤 SDÍLEJTE OKAMŽITĚ
Vyberte soubor přímo v aplikaci nebo použijte funkci „Sdílet" z jakékoli jiné aplikace v Androidu. EasyShare nahraje soubor na váš server a vygeneruje krátké URL, které můžete poslat komukoli.

🔗 KRÁTKÉ ODKAZY
Každý sdílený soubor získá unikátní krátkou URL adresu (např. https://vas-server.cz/xK7mPq2n). Příjemce nepotřebuje žádný účet — stačí odkaz a může soubor stáhnout.

⏳ PLATNOST ODKAZŮ
Nastavte, jak dlouho mají vaše sdílené odkazy platit. Na výběr je 1 hodina, 24 hodin, 7 dní, 30 dní nebo bez omezení. Prošlé odkazy se automaticky odstraní.

📋 AUTOMATICKÉ KOPÍROVÁNÍ DO SCHRÁNKY
Po nahrání se odkaz automaticky zkopíruje do schránky. Stačí ho jen vložit a odeslat.

🔒 VLASTNÍ SERVER = SOUKROMÍ
Vaše soubory zůstávají na VAŠEM serveru. Žádné cloudové úložiště třetích stran. Vy rozhodujete, kdo může nahrávat (ověření API klíčem) a kde se data ukládají.

⚙️ SNADNÉ NASTAVENÍ
Průvodce prvním spuštěním: stačí zadat URL serveru, API klíč a můžete sdílet. Konfiguraci lze importovat přímo z webového průvodce na serveru.

📲 IMPORT A EXPORT KONFIGURACE
Importujte konfiguraci ze stránky nastavení serveru nebo exportujte nastavení pro přenos na jiné zařízení.

🌐 ČEŠTINA I ANGLIČTINA
Aplikace automaticky detekuje jazyk zařízení a zobrazí rozhraní v češtině nebo angličtině.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

POŽADAVKY:
• Vlastní webový server s PHP (IIS nebo Apache)
• Nainstalovaná serverová část EasyShare (zdarma, open source)
• Připojení k internetu

EasyShare je open source. Serverová část a Windows klient jsou k dispozici na:
https://github.com/user/EasyShare
```
