## Full Description (max 4000 chars)
```
EasyShare is a self-hosted file sharing app that connects to your own web server. Upload any file and get a short shareable link — no cloud services, no file size limits, full control over your data.

📤 SHARE FILES INSTANTLY
Select a file or use Android's Share feature to send files directly from any app. EasyShare uploads the file to your server and gives you a short URL link you can send to anyone.

🔗 SHORT SHARE LINKS
Every shared file gets a unique short URL (e.g. https://your-server.com/xK7mPq2n). No accounts needed — anyone with the link can download the file.

⏳ LINK EXPIRATION
Set how long your shared links stay active. Choose from 1 hour, 24 hours, 7 days, 30 days, or unlimited. Expired links are automatically removed.

📋 AUTO-COPY TO CLIPBOARD
After upload, the share link is automatically copied to your clipboard. Just paste and send.

🔒 SELF-HOSTED & PRIVATE
Your files stay on YOUR server. No third-party cloud storage. You control who can upload (API key authentication) and where data is stored.

⚙️ EASY SETUP
One-time setup wizard: enter your server URL, API key, and you're ready to go. Configuration can be imported from the server's web setup page.

📲 CONFIGURATION IMPORT/EXPORT
Import configuration from your server's setup page or export settings to transfer to another device.

🌐 MULTIPLE LANGUAGES
The app automatically detects your device language and shows the interface in Czech or English.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

REQUIREMENTS:
• Your own web server with PHP (IIS or Apache)
• EasyShare server component installed (free, open source)
• Internet connection

EasyShare is open source. Server component and Windows client available at:
https://github.com/user/EasyShare
```
